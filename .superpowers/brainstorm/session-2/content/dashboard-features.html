Diseño del Cotizador: Tablero Maestro (Single Screen)

Todo se ajusta en tiempo real mediante controles reactivos. ¿Qué elementos principales incluimos?

A

Selector de Ubicación

Mapa interactivo o selector de estado para ajustar la radiación solar real según la zona de México.

B

Visualizador de Techo 3D

Una representación 3D simple que añade o quita paneles a medida que el usuario ajusta el sistema.

C

Gráfica de Retorno (ROI)

Gráfica dinámica que muestra en cuántos años se paga el sistema y el ahorro a 25 años.

D

Boton "Descargar Memoria Técnica"

Un CTA al final del tablero para enviar el resumen del cálculo al correo del usuario.
