¿Qué estilo de inmersión visual (3D/Scroll) buscamos para eSol?

He investigado las tendencias 2024-2025 para empresas de energía solar (sitios galardonados en Awwwards). Estas son las 3 rutas principales. Haz clic en la que más te guste.

A

El Recorrido Narrativo (Scroll Storytelling)

Enfoque: Énfasis en animaciones vinculadas al scroll (Parallax multicapa). Los elementos (paneles, textos, fondo) se mueven a distintas velocidades. El usuario "viaja" por el proceso solar a medida que hace scroll.

  • Perfecto para explicar el Anteproyecto 3D paso a paso.
  • Tecnología: GSAP (ScrollTrigger) + CSS Parallax.
  • Rendimiento: Muy bueno y fluido en móviles.
B

Modelos 3D Interactivos (WebGL)

Enfoque: En lugar de imágenes planas, integramos modelos 3D reales (ej. un panel solar o un inversor) que el usuario puede rotar con el ratón. Esto da una vibra tecnológica ("Tech/Premium").

  • Ideal para mostrar los "Productos de Primera Línea".
  • Tecnología: Three.js o Spline3D.
  • Rendimiento: Requiere optimización cuidadosa para no afectar el SEO.
C

Bento Grid "Glassmorphism" + Micro-interacciones

Enfoque: Un diseño altamente modular (cajas con bordes redondeados tipo Apple/Tesla) sobre fondos oscuros desenfocados. Interacciones sutiles al pasar el cursor (brillos, contadores que suben).

  • La ruta más limpia y moderna para mostrar datos técnicos sin abrumar.
  • Tecnología: Vanilla CSS + Framer Motion / animaciones CSS.
  • Rendimiento: Excelente, el "sweet spot" entre moderno y rápido.
